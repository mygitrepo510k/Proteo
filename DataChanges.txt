On BlueSphere the following values need to be set, using the correct Customer ID, for ApplicationProfile table
- DeviceCheckInOutRequired : to be set true for hauliers those will require their devices to be checked out and in. Else false.
- DeviceManagementAPIURL : Set to http://<ProteoAnalyticsProductionURL>/api/devicemanagement/
- DeviceCheckOutTermsAndConditions : Set the Terms and conditions to be accpted for using the haulier's device
